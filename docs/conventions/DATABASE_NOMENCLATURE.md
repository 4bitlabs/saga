## Constraint Naming Convention

- **Primary Key:** `PK_{TABLE_NAME}`
- **Foreign Key:** `FK_{COLUMN_NAME}_{PARENT_TABLE_NAME}_{CHILD_TABLE_NAME}`
- **Not Null:** `NN_{COLUMN_NAME}_{TABLE_NAME}`
- **Unique:** `UN_{COLUMN_NAME}_{TABLE_NAME}`
- **Index:** `IDX_{MOTIVE}_{COLUMN_NAME}_{TABLE_NAME}`
